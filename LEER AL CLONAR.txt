BUENO, AQUI DIRÉ LO QUE ES NECESARIO TENER YA INSTALADO POR SI DE PRONTO ALGUIEN AJENO AL PROYECTO LO CLONA O SI PASA MUCHO TIEMPO
Y NO ME ACUERDO DE COMO INICIALIZARLO, ES NECESARIO TENER INSTALADO MYSQL, XAMPP CON EL MYSQL WORKBENCH. 
CUANDO LO VAYAN A INICIAR, ABREN EL XAMPP Y PRENDEN APACHE Y MYSQL, POR AHÍ DERECHO LE DAN EN ADMIN A AMBOS. DESPUES SIGUE CREAR
LA BASE DE DATOS Y LAS TABLAS, PARA ESO EN LA CARPETA DEL PROYECTO HAY UN ARCHIVO .SQL QUE SE DEBE IMPORTAR AL PHPMYADMIN
PARA QUE AUTENTIQUE TODA ESA COSA, O AL WORKBENCH, EL QUE QUIERAN USAR, ESE LADO BIEN.
ANTES DE TRABAJAR ENCIMA DEL PROYECTO, MUEVAN LA CARPETA QUE CONTIENE EL PROYECTO A LA SIGUIENTE RUTA QUE CASI SIEMPRE POR
DEFECTO ES: C:\xampp\htdocs, PARA QUE CUANDO TENGAN QUE HACER OPERACIONES, CAMBIOS U OTRA CLASE DE VAINAS NO SALGAN EN ERROR.
ORGANIZEN BIEN EL PHP EN VISUAL STUDIO CODE (O AL MENOS A MI ME TOCÓ INGRESAR A GITHUB PARA QUE ESO CUADRARA ALGO QUE PEDÍA).
